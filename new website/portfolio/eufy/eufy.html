Make Eufy smart again!

  • Category: DIY
  • Project date: Fall 2019
  • Project URL: Make-Eufy-Smart-Again

Make a non-internet controlled vacuum 'smart'

I have an old vacuum cleaner that works fine, but did not have Wifi capability, only an IR remote control. This project copies the remote codes using an arduino based IR library, and uses a IR blaster connected to esp8266 that is itself throned on the vacuum to control the robot. It even includes plans for Blynk - so you can have GUI control over the vacuum cleaner.
